O que é um HTML?

HTML significa HiperText Markup Languege, mais conhecido como uma linguagem de marcação, ele é usado para criar uma estrutura básica de um website, onde consegue deixar organizado os elementos de uma página web, ou seja, sem isso o navegador não conseguiria mostrar textos como elementos ou carregar imagens.

Para melhor entender através do HTML há um documento de extensão .html, que o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa vizualizá-lo.


Como é uma estrutura básica de um HTML?

O HTML é formado por tags, sua estrutura básica é composta por uma tag doctype(definição do documento), por um head(cabeçalho) e por um body (corpo).
Doctype - Definindo o documento: O doctype não é uma tag HTML, mas uma instrução para o navegador e outros programas que podem ler seu site, que o código encontrado ali é um código HTML. O doctype é obrigatório e na estrutura do codigo ele deve ser sempre na primeira linha do seu documento.
HEAD - Cabeçalho: São informações que não são transpostas visivelmente para o usuário São dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados
BODY - CORPO: Trata-se do documento em si, ou seja, a informação legível para o usuário. É todo e qualquer texto que se deseja apresentar, assim como toda e qualquer forma de mídia de saída (imagens, sons) e toda a apresentação de entrada de dados como formulários também se aplica neste seção do documento.
